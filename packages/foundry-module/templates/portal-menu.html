Portail de création de personnage

Les joueurs remplissent leur fiche sur le portail web (service externe). Les soumissions arrivent en file d'attente ; ce module les importe comme acteurs dnd5e dans le monde. L'import s'effectue dans votre navigateur : gardez Foundry ouvert.

Connexion au portail

Le module interroge le portail à cet intervalle quand il est activé.

Catalogue SRD

Publie les races / classes / historiques / sorts / objets du système vers le formulaire.

Lien joueur (usage unique)

Si choisi, le personnage créé devient le perso par défaut de ce joueur (+ OWNER).
